# Introduction

Power Calculator helps users convert and understand the conversion between any vendor's concept of "transmit power" to any other vendor's transmit power.

This includes infrastructure vendors such as Cisco, Aruba, Juniper, and Fortinet, and planning vendors such as Hamina.

# Data Source

All of the research for how vendors display transmit power was by Jerry Olla. His researched was used as the basis for this tool.

# Reliabiltiy

This tool is currently completely and totally untested. The output has not been validated at all. Use with extreme skeptisicm.
